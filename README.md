# Counter App

A simple counter application built with HTML, CSS and JavaScript.

## Features
- Increase counter
- Decrease counter
- Reset counter
- Dynamic color change based on counter value

## Technologies Used
- HTML
- CSS
- JavaScript (DOM & Events)

## Project Structure

```
counter-app
│
├ index.html
├ app.css
└ app.js
```
